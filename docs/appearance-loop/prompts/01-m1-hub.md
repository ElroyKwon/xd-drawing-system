# M1 메타프롬프트 — Hub 표면 (executor contract)

> ai-loop Phase 2.5 산출물. 이 텍스트는 **freeze**됨(2026-06-24, 3개 결정 사용자 확정). 구현 에이전트는 이 지시에 맞춰 실행하고, 별도 검증팀은 아래 **Acceptance checklist**를 항목별로 채점한다. 합격 위해 이 기준을 몰래 고치지 않는다(기준 변경 = 스코프 변경 = HUMAN_GATE).

## Stage goal

이미 존재하는 Hub 셸(`src/App.tsx`)을 카탈로그(§A My Home / §B 프로젝트 템플릿) 구조 수준으로 끌어올린다. 대상: **My Home**, **프로젝트 템플릿**. 프로젝트 목록은 이번 범위 제외. "없는 화면 만들기"가 아니라 카탈로그가 기록한 영역·컨트롤·빈상태·탭·모달의 **갭 메우기**.

## Co-design log (frozen decisions — 2026-06-24)

1. **상호작용 깊이 = 구조 우선.** 카탈로그에 보이는 영역·카드·빈상태·탭·모달 구조를 모두 갖춘다. 이미 동작하는 것(탭 전환, 모달 열고닫기, 2단계 템플릿 작성 플로우)은 유지. 필터칩·페이지네이션 등 부가 컨트롤은 **외관(비활성) 허용** — 살아있는 토글/핸들러를 새로 강제하지 않는다.
2. **샘플 템플릿 카드 주 액션 = 작성 모달 열기.** 카드의 "사용하여 생성" 클릭 시 **프로젝트 작성 모달**을 열고 템플릿 드롭다운을 해당 템플릿으로 프리필한다(순수 affordance, 영속화 없음).
3. **프로젝트 목록 미손댐.** 카탈로그상 이미 "구현됨"(~85%). 액세스 드롭다운·행 케밥·페이지네이션 활성화는 후속 마일스톤으로 미룬다.

## Instruction (실행 지시)

`src/App.tsx`(+ 필요 시 `src/styles.css`)만 편집한다. 아래 갭을 카탈로그·원본 캡처 기준으로 메운다.

### My Home (`MyHomeView`, §A / 캡처 173732·173742)
- 온보딩 배너(`showTour`): "Take the tour" + "Learn more" + 닫기(X) 유지. **이미 충족 — 회귀만 방지.**
- 할당 위젯: 필터칩 3종("나에게 할당됨"·"내 회사에 할당됨"·"내 액션에 지정됨", 첫 칩 활성 표시) + "No assignments found" 빈상태 + "필터 지우기". **이미 충족 — 회귀 방지.**
- 지도 위젯: 정적 플레이스홀더(MapPin) 유지. **외부 지도 연동 금지(HUMAN_GATE).**
- 책갈피 위젯: 편집(연필) 아이콘 + "아직 북마크가 없습니다" 빈상태 + 추가 링크. **이미 충족 — 회귀 방지.**
- 최근 항목 위젯: 테이블(이름·마지막으로 연 날짜·프로젝트 이름·허브 이름) + 5개 샘플 행 + 페이저 라벨. **이미 충족 — 회귀 방지.**
- 그리드 레이아웃: 2열(할당 | 지도) + (책갈피 | 최근항목 전폭에 가까운 배치)이 캡처와 같은 의미 구조인지 확인. 깨지면 보정.

### 프로젝트 템플릿 (`ProjectTemplatesView`, §B / 캡처 173312·173328·173339)
- **[갭] "샘플 템플릿" 섹션 접기/펴기 헤더** 추가 — 캡처 173312의 `∨ 샘플 템플릿` collapsible 헤더. 기본 펼침. 접기 토글은 로컬 state 1개 허용(구조 표현용, "구조 우선" 예외 아님 — 섹션 자체가 카탈로그 영역).
- 샘플 카드 4종(General Contractor / Public Service Owners / Investment Owners / Owner Operator): 이름 + 액세스 칩 + **주 액션 버튼**. **[갭] 주 액션 = "사용하여 생성"** 버튼을 추가하고, 클릭 시 프로젝트 작성 모달을 열며 템플릿 필드를 해당 템플릿명으로 프리필(결정 2).
- "모든 보기" 버튼: 유지(비활성 허용).
- 허브 템플릿 섹션: "+ 프로젝트 템플릿 작성" + 검색 + 필터(비활성 허용) + 빈상태/테이블 분기. **이미 충족 — 회귀 방지.**
- 2단계 작성 모달(유형 선택 → 이름 입력): 유지. **이미 충족 — 회귀 방지.**

### 프로젝트 작성 모달 (`ProjectCreateModal`)
- 템플릿 드롭다운이 외부에서 프리필될 수 있도록 최소 연결만 추가(샘플 카드 → 모달 오픈 시 초기 템플릿값 전달). 그 외 폼은 손대지 않는다.

## Acceptance checklist (검증팀이 항목별 PASS/FAIL)

**구조(카탈로그 커버리지)**

- [ ] A2. 프로젝트 템플릿: "샘플 템플릿" 접기/펴기 헤더가 존재하고 기본 펼침.
- [ ] A3. 샘플 카드 4종 각각에 이름·액세스 칩·"사용하여 생성" 주 액션 버튼 렌더.
